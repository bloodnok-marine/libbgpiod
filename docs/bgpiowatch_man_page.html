Man page for bgpiowatch

NAME

bgpiowatch - monitor gpio line configuration

SYNOPSIS

bgpiowatch [OPTIONS] <chip-id> <line-id>...

DESCRIPTION

Watch GPIO lines for reservation and configuration changes.

OPTIONS

-h, --help:

display this help message.

-q, --quiet:

execute quietly

-r, --repeat=count:

how many times to fetch (default=1)

-t, --timeout=millisecs

Specify an inactivity timeout period.

-v, --version:

display the version.

-x, --exec=path:

command to execute on event

Gpiochip-ids may be a full path to the gpiochip device, or an abbeviated suffix (eg "chip0") of a valid path.

Line-ids are integer line numbers. Commands specified by --exec will be passed the chip path, the line number, an event description and the event timestamp. A repeat count of zero means repeat forever. The result of the command will be 0, or the value of the last executed script.
