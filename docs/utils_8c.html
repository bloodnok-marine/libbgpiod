◆ path_for_arg()

char* path_for_arg ( svector *  paths,
char *  arg 
)

Return the full path for the gpio character device given by arg.

Parameters
pathssvector containing full paths to all gpio character devices. This may be NULL in which case path_for_arg() will call get_chip_paths(). The matching string from this struct will be the returned string. The caller may not free it and should ensure that the svector it comes from is not freed while the string is still in use.
argThe, possibly abbreviated, string identifying the gpiochip device in question. To match "/dev/gpiochip0", arg may contain: "gpiochip0", "chip0", or just "0".
Returns
The matching string from paths. See the note above for safety information on this result. If no match is discovered the program will abort with exit().
