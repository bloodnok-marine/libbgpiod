 bgpiodetect. Executable to identify and describe the gpio chips that are available. This is just like gpiodetect but uses libbgpiod instead of libgpiod. The big difference between the 2 libraries is that libbgpiod uses V2 system calls, where libgpiod (currently) uses the deprecated V1 calls.
