NAME

bgpioinfo - manual page for bgpioinfo 0.1.0

SYNOPSIS

bgpioinfo [OPTIONS] [gpiochip id] [line-no]...

DESCRIPTION

List information about gpio lines for gpio chips. If no chip is specified, list for all chips. If no lines are specified list all lines.

OPTIONS

-h, --help:

display this help message.

-v, --version:

display the version.

gpiochip ids may be a full path to the gpiochip device, or an abbeviated suffix (eg "chip0") of a valid path.
