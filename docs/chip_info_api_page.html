Getting Information On Specific GPIO Lines

Information about specific GPIO lines can be obtained from the gpio_v2_line_info struct returned from the get_lineinfo() function.

The full set of steps to do this is:

  • call open_gpio_chip() to open the device;
  • then, for each gpio line:
    • call get_lineinfo();
    • examine the returned gpio_v2_line_info struct, directly and using:
      • bgpio_attr_flags();
      • bgpio_attr_output();
      • and bgpio_attr_debounce();
    • free the struct;
  • call close_gpio_chip() to close the device.

The command line tool bgpioinfo (source file bgpioinfo.c) provides a full working example of this.

A simpler example can be found in info.c in the examples directory. This is the main function from that file:
