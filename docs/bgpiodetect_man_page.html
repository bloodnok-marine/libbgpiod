Man page for bgpiodetect

NAME

bgpiodetect - list gpio chips

SYNOPSIS

bgpiodetect [OPTIONS] [gpiochip-id]...

DESCRIPTION

List GPIO chips, their labels and the the number of lines.

OPTIONS

-h, --help:

display this help message.

-v, --version:

display the version.

gpiochip-ids may be a full path to the gpiochip device, or an abbeviated suffix (eg "chip0") of a valid path.
